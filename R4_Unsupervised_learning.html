Using basic inference on the above data we can conclude that :

  • Car Type 0 contains saloon version of cars with a high displacement, power, acceleration and milage while being heavy.

  • Car Type 2 contains the heaviest vehicles, probably SUVs with poor milage and acceleration but has the highest displacement and power
